SpringKart is a backend application for an e-commerce platform built using Spring Boot.
It provides RESTful APIs for managing products, carts, orders, and payments with a focus on scalability and secure data handling.

---

## Tech Stack

- Java
- Spring Boot
- MySQL
- REST APIs
- Docker
- Kubernetes

---

## Features

- Product management (create, update, delete, view)
- Cart and order processing workflows
- RESTful APIs for e-commerce operations
- Secure and consistent transactional data handling
- Containerized application using Docker
- Deployed on Kubernetes for scalability and fault tolerance

---

## Architecture Overview

- Monolithic Backend Application
- REST APIs exposed using Spring Boot
- MySQL used as the relational database
- Docker used for containerization
- Kubernetes used for deployment and lifecycle management
